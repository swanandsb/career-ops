☀️ Every morning (5 min)
1
Open a new Terminal window
The regular Terminal app on your Mac. Not Cursor.
2
Check if the scan ran overnight
cat ~/Documents/GitHub/career-ops/logs/scan.log | tail -20
If you see "New offers added: X" — great, new jobs were found. If you see errors — the scan had trouble, run it manually in step 4.
🎯 When you want to apply (15-30 min)
3
Open Claude Code in the career-ops folder
cd ~/Documents/GitHub/career-ops && claude
Wait for the prompt to appear. This is your AI job assistant.
4
Run the pipeline to score today's jobs
/career-ops pipeline
This reads all the new job links and gives each one a score out of 5. Takes 15-20 minutes. You can do other things while it runs.
5
Look at scores — only care about 4.0 or higher
When pipeline finishes it shows a table. Only apply to roles that scored 4.0/5 or above. Everything below that — ignore it.
6
Generate a tailored resume for each good role
Paste the job URL directly into Claude Code:
https://jobs.ashbyhq.com/company/job-id-here
It will make a PDF just for that company. Wait for it to finish before pasting the next URL.
7
Apply manually at each job link
Open the job URL in your browser. Upload the PDF from the output/ folder. The file is named something like cv-swanand-barsawade-anthropic-2026-04-28.pdf
⚠️ The computer never applies for you. You always click Submit yourself.
🔁 If scan didn't run automatically
8
Run scan manually in Claude Code
/career-ops scan
This checks 70+ companies for new job postings matching your profile. Takes about 2 minutes.
