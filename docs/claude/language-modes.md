# Language Modes Reference

Default modes are in `modes/` (English). Switch when:
- User says "use [language] modes"
- User sets `language.modes_dir` in `config/profile.yml`
- You detect a JD in that language → suggest switching

## Available Languages

**German (`modes/de/`)** — DACH market
Files: `_shared.md`, `angebot.md` (eval), `bewerben.md` (apply), `pipeline.md`
Vocabulary: 13. Monatsgehalt, Probezeit, Kündigungsfrist, AGG, Tarifvertrag

**French (`modes/fr/`)** — France/Belgium/Switzerland/Luxembourg
Files: `_shared.md`, `offre.md` (eval), `postuler.md` (apply), `pipeline.md`
Vocabulary: CDI/CDD, SYNTEC, RTT, mutuelle, prévoyance, 13e mois, intéressement, RTT, titres-restaurant, CSE

**Japanese (`modes/ja/`)** — Japan
Files: `_shared.md`, `kyujin.md` (eval), `oubo.md` (apply), `pipeline.md`
Vocabulary: 正社員, 業務委託, 賞与, 退職金, みなし残業, 年俸制, 36協定, 通勤手当, 住宅手当

**Portuguese (`modes/pt/`)** — Brazil/Portugal
Files: `_shared.md`, `oferta.md` (eval), `aplicar.md` (apply), `pipeline.md`

**Russian (`modes/ru/`)** — Russian-speaking market
Files: `_shared.md`, `oferta.md` (eval), `apply.md`, `interview-prep.md`, `pipeline.md`

Use default English modes for English-language roles even at non-English companies.
